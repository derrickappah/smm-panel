# Documentation

This directory contains all project documentation organized by category.

## Directory Structure

### `setup/`
Setup and installation guides for different components.

- **README_SETUP.md** - Quick setup guide
- **SETUP.md** - Backend setup instructions
- **SUPABASE_SETUP.md** - Supabase configuration guide
- **VERCEL_SETUP.md** - Vercel deployment and environment variables
- **SMMGEN_SETUP.md** - SMMGen API integration setup

### `troubleshooting/`
Troubleshooting guides for common issues.

- **TROUBLESHOOTING.md** - General troubleshooting guide
- **TROUBLESHOOTING_DATABASE.md** - Database-specific troubleshooting

### `admin/`
Admin-specific documentation.

- **ADMIN_RLS_FIX.md** - Guide for fixing admin Row Level Security issues

### Root Files
- **ISSUES_FIXED.md** - Log of issues that have been resolved
- **test_result.md** - Test results and reports

## Quick Links

- **New to the project?** Start with `setup/README_SETUP.md`
- **Deploying to Vercel?** See `setup/VERCEL_SETUP.md`
- **Having issues?** Check `troubleshooting/` directory
- **Admin problems?** See `admin/ADMIN_RLS_FIX.md`
